# PRODIGY_DS_02

## Task 02 - Data Cleaning and Exploratory Data Analysis (EDA)

This project is part of my Data Science Internship at Prodigy InfoTech.

### Objective
Perform data cleaning and exploratory data analysis (EDA) on the Titanic Dataset to explore relationships between variables and identify patterns and trends.

### Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

### Data Cleaning Performed
- Filled missing values in Age using median values
- Filled missing values in Embarked using mode
- Removed the Cabin column due to excessive missing values

### Exploratory Data Analysis
- Survival distribution analysis
- Survival based on gender
- Survival based on passenger class
- Age distribution analysis
- Correlation heatmap

### Key Insights
- Female passengers had higher survival rates than males.
- First-class passengers had better survival chances.
- Most passengers were between 20–40 years old.
- Passenger class and fare influenced survival outcomes.

Titanic Dataset
